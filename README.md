A first installation of WCsim is required to run any of those files. This can be done with a software container and is well documented in https://github.com/WCTE/SoftwareContainer. 

The following description provides the tips for running comparing WCTE data with WCsim.//
** A mapping needs to be applied between WCTE PMT and WCSim
** Make sure to reference the correct source file present in WCSim/build/this_wcsim.sh
** The WCSim executable is in WCSim/build/install/bin/WCSim and needs the macro and tuning parameters
